# Plantillas de Word

Esta carpeta está destinada a almacenar plantillas de documentos de Word que los usuarios pueden subir y editar.

## Cómo usar las plantillas

1. **Subir plantilla**: Los usuarios pueden subir sus propias plantillas de Word (.doc o .docx) desde la interfaz
2. **Editar contenido**: Una vez subida, la plantilla se puede editar usando el editor integrado
3. **Generar PDF**: El documento editado se convierte a PDF con la foto insertada automáticamente

## Formatos soportados

- `.doc` - Documentos de Word 97-2003
- `.docx` - Documentos de Word 2007 y posteriores

## Límites

- Tamaño máximo: 10MB
- Solo archivos de Word válidos

## Funcionalidades del editor

- Formato de texto (negrita, cursiva, subrayado)
- Alineación de texto
- Campos dinámicos: [NOMBRE], [FECHA], [DOCUMENTO]
- Inserción automática de foto capturada
- Conversión a PDF profesional

## Estructura recomendada para plantillas

Para mejores resultados, las plantillas deberían incluir:

- Campos claramente marcados para información personal
- Espacio designado para la foto (se insertará automáticamente)
- Estructura clara y profesional
- Marcadores dinámicos para campos que cambian frecuentemente
